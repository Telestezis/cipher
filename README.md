1️⃣ ПРОЦЕНТНАЯ СТАВКА
Определения
Термин
Определение
Процентная ставка (i)
Отношение процентов за год к сумме ссуды
Ссуда (P)
Передача денег/ценностей на условиях возврата с процентами
Доход от ссуды (I)
Денежные средства от предоставления в долг
Наращенная сумма (S)
S = P + I
Виды процентных ставок
Вид
Обозначение
Формула
Простая
i
S = P(1 + ni)
Сложная
i*
S = P(1 + i*)ⁿ
Номинальная
j
S = P(1 + j/m)ᵐⁿ
Сила роста
δ
S = Pe^(δn)
Брутто-ставка
r
r = i + инфляционная премия
Эквивалентность ставок
Простая и сложная: 1 + ni = (1 + i)ⁿ*
Сложная и номинальная: (1 + i)ⁿ = (1 + j/m)ᵐⁿ*
Эффективная ставка: i = (1 + j/m)^m − 1*

3️⃣ ПОГАШЕНИЕ ЗАДОЛЖЕННОСТИ
Основные понятия
Задолженность = основной долг + начисленные проценты
Рента — задолженность, получаемая с должника
Контур платежей — график выплат (должен быть замкнутым, остаток = 0)
Виды платежей
Вид
Характеристика
Формула
Аннуитетный
Равные платежи каждый период
R = P × (j/m) / [1 − (1 + j/m)^(-m)]
Дифференцированный
Платежи уменьшаются к концу срока
Rᵢ = P/m + %ᵢ(еж)
4️⃣ ДИСКОНТИРОВАНИЕ
Определение
Дисконтирование — приведение будущей стоимости денег к текущей (действие, обратное наращиванию)
Формулы дисконтирования
Тип
Формула
Без капитализации
P = S/(1 + ni)
С капитализацией
P = S/(1 + i*)ⁿ
Со ставкой дисконта
P = S/(1 + d)ᵗ
Ставка дисконтирования
Факторы влияния:
Качество актива/проекта (ликвидность, размер, стадия жизненного цикла)
Продолжительность периода прогнозирования
Валютный фактор
Инфляционные ожидания
Ключевая ставка Центробанка
Методы расчёта:
Метод безрисковой ставки
Интуитивный метод (безрисковая + инфляция)
Метод экспертной оценки
Метод рентабельности (ROE, ROA)
ИНВЕСТИЦИИ
Определение: Размещение капитала с целью получения прибыли
Признаки:
Способность приносить доход
Определённый срок вложения
Целенаправленный характер
Наличие риска
Классификация:
Критерий
Виды
По объектам
Реальные, Финансовые, Нематериальные
По собственности
Частные, Государственные, Иностранные, Совместные
По сроку
Краткосрочные (<1 года), Среднесрочные (1-3 года), Долгосрочные (>3 лет)
По риску
Низкий, Средний
Источники финансирования: внутренние/внешние, собственные/привлечённые/заёмные
ИНВЕСТИЦИОННЫЙ МЕНЕДЖМЕНТ
Определение: Деятельность по исследованию, анализу и регулированию инвестиционного рынка
Задачи:
Анализ и оценка инвестиционных объектов
Обеспечение роста предприятия
Максимизация доходности
Минимизация рисков
Финансовая устойчивость
Своевременная реализация программы
Функции: Планирование → Организация → Мотивация → Контроль
ОЦЕНКА ЭФФЕКТИВНОСТИ ИНВЕСТИЦИЙ
Статические методы (без дисконтирования)
Показатель
Формула
Критерий
PP (срок окупаемости)
ΣSᵢ > P
Чем меньше, тем лучше
ARR (норма прибыли)
Средняя прибыль / Средние инвестиции
Чем больше, тем лучше
Динамические методы (с дисконтированием)
Показатель
Формула
Критерий
NPV (чистая приведённая стоимость)
Σ(Sᵢ/(1+d)ⁱ) − P
> 0 — принять
IRR (внутренняя норма доходности)
IRR = d, при котором NPV = 0
> d — принять
DPP (дисконтируемый срок окупаемости)
Σ(Sᵢ/(1+d)ⁱ) ≥ P
Чем меньше, тем лучше
PI (индекс рентабельности)
Σ(Sᵢ/(1+d)ⁱ) / P
> 1 — принять
